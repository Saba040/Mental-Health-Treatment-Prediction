პროექტი მიზნად ისახავს მკურნალობის საჭიროების (Treatment) პროგნოზირებას ინდივიდის დემოგრაფიული და სოციალურ-ეკონომიკური მახასიათებლების საფუძველზე.
მოცემული პრობლემა არის ბინარული კლასი (“კი” / “არა”) და მოიცავს მონაცემთა დამუშავებას, ფუნქციების ინჟინერიას, მოდელირების შექმნას და შეფასებას.

გამოყენებული ტექნოლოგიები:
Python
Pandas, NumPy
scikit-learn (Logistic Regression, Decision Tree, Random Forest, Naive Bayes)
Matplotlib, Seaborn (Confusion Matrix ვიზუალიზაცია)

შედეგები

შექმნა ბინარული კლასიფიკაციის სრული პაიპლაინი ნედლი მონაცემებიდან პროგნოზამდე.
ავტომატიზირებული მონაცემთა დამუშავება, Missing Value Imputation და Encoding.
სხვადასხვა ML ალგორითმების შედარება Accuracy, Precision, Recall, F1-score მიხედვით.
Confusion Matrix-ების გამოყენება მოდელის შესრულების ხარისხის ვიზუალიზაციისთვის.
Logistic Regression, Decision Tree, Random Forest, Naive Bayes მოდელების გამოყენება საუკეთესო მოდელის ასარჩევად.
